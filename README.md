# MUSIM.CO

대학교 졸업 작품으로 진행한 팀 프로젝트입니다.  
Python Flask 기반의 웹 서비스를 팀 단위로 개발했습니다.

---

## 기술 스택
- Python  
- Flask  
- MySQL  
- Git  

---

## 협업 경험

- 기능 단위 브랜치 생성 및 PR을 통한 병합 진행
- 팀원 간 역할 분담 후 API 연동 작업 수행
- Git 사용 과정에서 브랜치 전략의 중요성을 경험

---

## 회고

졸업 프로젝트를 통해 협업 환경에서의 개발 과정을 경험했습니다.  
이후 개인 프로젝트에서는 Git 사용 방식과 설계 구조를 더 체계적으로 개선하고 있습니다.
